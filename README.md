# MERN AUTHENTICATION and CSV PARSING

## Getting Started

### Prerequisites

Node-js v16.16.x
</br>
MongoDB vlatest

### clone this repository
```
git clone https://github.com/ihtisham007/MERN_Authorization.git
```
### open directory
```
cd MERN_Authorization
```
### for backend
```
cd backend
npm i 
npm i -g nodemon
npm start
```
### for frontend
```
cd frontend
npm i 
npm start
```

Code is Running on 
+ http://localhost:3000/
